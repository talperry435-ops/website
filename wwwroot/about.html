About This App

this website is my personal project for the bagrut assignment, and it contains all the necessary components of a simple website.
i picked this idea because of a different app called "inkwell games" that i am very fond of and also has 3 distinct puzzle games, just of a higher difficulty level.

How it is built

Part What it does
HTML pages Provide the content for Home, About and games.
CSS file Keeps the layout clean and consistent without a complex design.
JavaScript file Handles the technical side of the website, mainly the games' interactivity, but also manages the server-side logic.
C# file Serves the pages and returns some data for the browser to display.

The JavaScript

the javascript in my website handles the interactive elements of the puzzle games, meaning the buttons the user can press and the interactive features in find the differences and find the object.
more specifically, if you press certain parts of the pictures, the javascript is responsible for switching the image to a different one, with the corresponding part circled red.

The server-side implementation

the c# backend is the one receiving the username inserted at the home page and is responsible for sending the number of differences back to the javascript when a button is pressed.

Built with HTML, CSS, JavaScript, and C# .NET Minimal API.
