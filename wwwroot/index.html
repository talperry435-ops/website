Welcome to the bagrut puzzle app!
here you can find all kinds of games and puzzles to challenge your mind and have some fun on the way!
the website contains 3 games, find the differences, a maze, and "find the object" (similar to wheres waldo).

How to play each game:

  • find the differences - Look for the differences between the two images, clicking on a difference will fresult in a red circle appearing around it.
  • maze - Navigate through the maze using your cursor to reach the exit.
  • find the object - Locate the specified object hidden in the scene, there are aq few of said object but all iof them will be hidden well within the scene.

what each page does:

page What it contains
home page explains about the website and its features, while also providing tips for the games.
about page Provides information about the technical side of the website and why it was made this way.
games page contains the core of the website - the actual games.

Tips for the games:

  • in the maze, sometimes starting from the end and going backwards can be more effective.
  • in the find the differences game, try to look for differences in the corners of the images first, as they are usually easier to spot.
  • in find the object game, try to scour the image with a certain approach instead of randomly looking around.
  • sometimes, zooming in on the image can help you see details you might have missed.
